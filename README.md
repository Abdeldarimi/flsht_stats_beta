# Statistiques Website (Enhanced Tailwind UX)

Instructions:
1. Put folder `statistiques-website-enhanced` into `htdocs` or `www` (XAMPP/WAMP/Laragon).
2. Import `database.sql` into phpMyAdmin to create the database and tables.
3. In the project folder run:
   ```bash
   composer require phpoffice/phpspreadsheet
   ```
4. Open http://localhost/statistiques-website-enhanced/login.php
Default admin creation: use create_admin.php to set admin password if desired.
